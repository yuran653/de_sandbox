# FAQ: Подключение к VPN и доступ к сервисам

## Скачайте клиент WireGuard
Скачайте клиент WireGuard с официального сайта: https://www.wireguard.com/install/

## Подключение к VPN
1. Установите клиент WireGuard на ваш компьютер
2. Получите файл конфигурации `.conf` от администратора сервера
3. Импортируйте файл конфигурации в клиент WireGuard
4. Подключитесь к VPN. После подключения вы увидите новый сетевой интерфейс

## Доступ к сервисам через VPN
После подключения к VPN вы можете получить доступ к следующим сервисам только через VPN-туннель

### VPS_1 (Airflow + PostgreSQL + MinIO) - 10.104.0.5
- **Airflow Web UI**:
  - http://10.104.0.5:8080
- **MinIO S3 Console**:
  - http://10.104.0.5:9001
- **MiniO S3 API endpoint**:
  - http://10.104.0.5:9000
- **PostgreSQL базы данных**:
  - Airflow Metadata DB: 10.104.0.5:5433
  - Datalake DB: 10.104.0.5:5432

### VPS_2 (ClickHouse) - 10.104.0.2
- **ClickHouse HTTP Interface**:
  - http://10.104.0.2:8123
- **ClickHouse Native Protocol**:
  - 10.104.0.2:9000

### VPS_3 (Spark + Jupyter) - 10.104.0.3
- **Spark Master Web UI**:
  - http://10.104.0.3:30080
- **Jupyter Lab**:
  - http://10.104.0.3:30888

**Название баз, логины и пароли: получите у администратора сервера**

**Важно**: Все остальные интернет-трафик будет идти через ваш обычный интернет-провайдер (split tunneling). Сервисы доступны только через VPN

## Устранение неполадок и иные вопросы по подключению
- Свяжитесь с администратором сервера

## Безопасность
- Никогда не передавайте файл `.conf` третьим лицам
- При отключении VPN доступ к сервисам будет заблокирован
