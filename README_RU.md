# Руководство по подключению к VPN (de_sandbox)

Следуйте этим шагам, чтобы настроить защищенное соединение на вашем устройстве.

### Шаг 1. Установка WireGuard

Скачайте и установите клиент WireGuard для вашей операционной системы:

* **Windows / macOS:** Скачать с [официального сайта WireGuard](https://www.wireguard.com/install/).
* **Android:** Скачать из [Google Play](https://play.google.com/store/apps/details?id=com.wireguard.android).
* **iOS:** Скачать из [App Store](https://apps.apple.com/us/app/wireguard/id1441195209).
* **Linux:** Установите через менеджер пакетов, введя команду,
    ```bash
    sudo apt install wireguard
    ```

---

### Шаг 2. Генерация ключей

Вам нужно создать пару ключей: **Приватный** (секретный) и **Публичный** (для администратора).

#### Для пользователей Linux / macOS (через терминал):
1.  Откройте терминал.
2.  Выполните команду:
    ```bash
    wg genkey | tee de_sandbox_private.key | wg pubkey > de_sandbox_public.key
    ```
3.  Это создаст два файла в текущей папке.

#### Для пользователей Windows (через приложение):
1.  Откройте WireGuard.
2.  Нажмите стрелочку рядом с кнопкой «Добавить туннель» -> **«Добавить пустой туннель»**.
3.  Программа автоматически сгенерирует `PrivateKey` и `PublicKey`. Скопируйте их в блокнот.

---

### Шаг 3. Передача данных администратору

1.  Выполните команду
    ```bash
    cat de_sandbox_public.key
    ```
    (или скопируйте `PublicKey` из окна Windows-клиента).
3.  **Отправьте эту строку администратору** (в мессенджере или по почте).
4.  ⚠️ **Важно:** Никогда не отправляйте никому свой `Private Key` (Приватный ключ).

---

### Шаг 4. Настройка конфигурации

Администратор отправит вам необходимые данные:
* Ваш IP-адрес
* IP-адрес сервера
* Публичный ключ сервера

Создайте текстовый файл с именем `de_sandbox.conf` (или вставьте этот текст в поле конфигурации приложения WireGuard) и заполните пропуски:

```ini
[Interface]
# Вставьте сюда ваш ПРИВАТНЫЙ ключ (из шага 2)
PrivateKey = <ВАШ_ПРИВАТНЫЙ_КЛЮЧ>

# IP-адрес, который выдал вам администратор (например, 10.10.0.5/32)
Address = <ВАШ_IP_ОТ_АДМИНА>/32

# DNS сервер (рекомендуемый для стабильности)
DNS = 1.1.1.1

[Peer]
# ПУБЛИЧНЫЙ ключ сервера (пришлет администратор)
PublicKey = <КЛЮЧ_СЕРВЕРА_ОТ_АДМИНА>

# Адрес сервера (пришлет администратор)
Endpoint = <IP_СЕРВЕРА>:51820

# Разрешить весь трафик через VPN
AllowedIPs = 0.0.0.0/0, ::/0

# Поддерживать соединение активным (каждые 25 секунд)
PersistentKeepalive = 25
